Creating a home server

Install ubuntu server on a computer

Download ubuntu!

Install programs

apt-get -y install mailutils ssh php apache2 netcat

Create user

useradd -s /usr/sbin/nologin mailman

Create home directory

mkdir /home/mailman

Filter traffic

ufw allow 80

ufw allow 443

Install crontab

apt-get -y install cron

Crucial! edit /etc/hosts and add:

YOUR_FLAG_SERVER_IP flag

Test ssh to see if it works (don't forget your private key):

ssh mailman@flag
